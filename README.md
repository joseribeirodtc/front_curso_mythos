# Página de Cadastros do Curso Mythos

- Uma vez clonado o repositório, no propt de comando, execute `yarn install` no diretório clonado para instalar todas as dependências

-no terminal, no diretório onde esse repositório foi clonado, execute: `yarn start`
a página irá abrir no seu browser na porta 3000
url:http://localhost:3000/

-agora é só navegar!
